# NotesApp

A text-based Java application to save and read notes using File I/O.

## Features
- Write notes to a text file
- Read all notes from the file
- Simple terminal-based UI

##  Technologies
- Java
- FileWriter, FileReader, BufferedReader
- VS Code
- Terminal
